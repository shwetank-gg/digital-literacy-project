# Summary
In module 1 we create a one‑page infographic using tools like Canva or Prezi to explain digital literacy. It highlights safe internet use, professional presence, and email etiquette. The goal is to design a clear visual resource that helps our peers quickly understand why digital literacy is essential.

---

In module 2 we create professional profiles on platforms like GitHub, LinkedIn, and Kaggle. Using these profiles, our academic curriculum vitae and aspirations for future work are both on display. This task prepares us for internships, networking, and opportunities in our careers by laying the groundwork of our online identity.


----

In module 3 we attempt some easy coding problems on those great exercise sites like CodeChef or HackerRank, still with screenshots as part of the response .Not everyone is adept at reading code without dottings and color-coding!At the same time, we develop a quiz on digital literacy using Google Form.This in turn improves our technical problem-solving skills as well as our ability to collaborate using cloud tools even if we are working from home.

---

After we have read the module, you can draft two professional emails–one to a professor and the other to an internship coordinator–with excellent structure and tone. We also prepare a checklist for social media do’s and don’ts. Try to make sure that what we emphasized in this module–respectful communication and responsible online behavior–remains with you for a long time to come.
